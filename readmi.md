# SecuTrack: An IoT-Based Smart Security Monitoring and Intrusion Detection System with Realtime Web and Mobile Applications

Purpose:

- To detect human presence in unauthorized and restricted area such as houses, schools, offices, and other facilities without proper security, and to help prevent possible intrusions.

Problem:

- Many houses, schools, and offices become victims of theft and unauthorized access due to the lack of security personnel, limited monitoring systems, and delayed response during incidents.

Solution:

- The solution is to develop an Arduino-based smart security monitoring system that integrates multiple sensors and alert mechanisms for effective real-time monitoring

Our system provides 24/7 real-time sensor-based intrusion with RFID identity verification, user authentication, intelligent alerting, and full web and mobile monitoring, ensuring continuous protection of restricted areas.
